# 什么是好代码？

本库的目标在于写"笨代码"。
其间有两个要素，清晰的代码结构与完善的代码文档。各个目录下都有与此目录相关的README.md，来标明需要进行阅读的材料等。

# 深度学习一般套路

1. 加载数据
2. 构造输入
3. 搭建网络
4. 计算输出
5. 计算loss
6. 反向传播
7. 打完收工

# 环境依赖

主目录下执行：pipreqs . --encoding=utf-8 --force，生成requirements文件

本地执行：pip install -r requirements，安装所有的依赖
